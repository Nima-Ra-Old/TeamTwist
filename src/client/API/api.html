Get Started

For using Team Twist's API you should send requests with the following structure:
https://TeamTwist.com/api/MethodName

Get Your User Id

For using Team Twist's API you should have user id to pass as a parameter for post method and you can get it from settings of your panel:
https://TeamTwist.com/panel

getUser()

Note: The following content is not real and it is just a test case.

id Integer 123

changeEmail()

You can change your account email with this method. Parameters are listed below:

Parameter Type Required Description
user_id String Yes No description
new_email String Yes The new email address which you want to be your email address

changeName()

You can change your account name with this method. Parameters are listed below:
Note: this method changes your account name, not your account username. The username method is changeUsername()

Parameter Type Required Description
user_id String Yes No description
new_name String Yes The new name address which you want to be your name address

changePassword()

You can change your account password with this method. Parameters are listed below:

Parameter Type Required Description
user_id String Yes No description
new_password String Yes Your password should be at least 8 digit password
old_password String Yes For validation you need to send your old password

changePhone()

You can change your account phone number with this method. Parameters are listed below:

Parameter Type Required Description
user_id String Yes No description
new_phone String Yes You must include your country code. Example: +98123456789

changePhoto()

You can change your account photo with this method. Parameters are listed below:

Parameter Type Required Description
user_id String Yes No description
new_photo URL Yes You should pass a url or change your profile photo from your panel.

changeUsername()

You can change your account username with this method. Parameters are listed below:

Parameter Type Required Description
user_id String Yes No description
new_username String Yes No description

getProjects()

Parameter Type Required Description

getStatistics()

You can get your projects, teams, todo and so on's statistics with this method. Parameters are listed below:
You should pass one of those optional items.

Parameter Type Required Description
user_id String Yes No description
team String Optional The team name. If you dont pass team you should pass another one optional item.
project String Optional The project name. If you dont pass project you should pass another one optional item.

You can signup new users with this method. Parameters are listed below:

Parameter Type Required Description

password String Yes No description
